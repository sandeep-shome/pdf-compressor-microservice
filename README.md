# PDF Compressor Microservice

A PDF compression system that builds on the microservice architecture, in a very loosely coupled manner.

## Features

- 🏗️ Up to 75% PDF size reduction

- ⛓️‍💥 De-coupled System

- ⚡ Lightning Fast Speed

- 🛠️ Native Compression Quality Slider

- 📱 Cross platform

## Tech Stack

- **Docker** - Managing Images & containers, Orchestration

- **Node.js** - File-Manager Service

- **Python** - Compression & Janitor Service

- **GhostScript** - System level compression tool

- **MongoDB** - Document DB for File Meta Data

- **RabbitMQ** - Queue File Compression Jobs

- **Minio** - Object Storage For PDF Files

- **Nginx** - Reverse Proxy & Web Server

- **React** - Front-end SPA

- **TailwindCSS** - UI Designing
