# Vite React App

Front-end Service for the PDF Compression App.

## Features

- React Router

- React Hot Toast

- React Spinners

- React

- Vite

- TailwindCSS

- Typescript

## Run Locally

Install dependencies

```bash
pnpm install
```

Start the server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

- `VITE_BASE_URL` - URL of the API server
